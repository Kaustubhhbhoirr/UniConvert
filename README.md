## 📌 About
UniConvert is a lightweight, fully offline desktop unit converter application developed using Java Swing. It supports eight conversion categories accessible from a single modern dark-themed main menu, with live real-time conversion as you type.

---

## 🗂 Categories
| # | Category | Units |
|---|----------|-------|
| 1 | Length | Angstrom, nm, µm, mm, cm, m, km |
| 2 | Temperature | Celsius, Fahrenheit, Kelvin |
| 3 | Speed | m/s, km/h, mph, knot |
| 4 | Number System | Decimal, Binary, Hex, Octal |
| 5 | Area | mm², cm², m², km², in², ft² |
| 6 | Volume | mL, cL, dL, L, m³, gal, qt, pt |
| 7 | Power | Watt, kW, MW, HP, BTU/hr, kcal/hr |
| 8 | Weight | mg, g, kg, Tonne, lb, oz, Stone |

---

## ✨ Features
- ⚡ Live real-time conversion using DocumentListener
- 🎨 Modern dark-themed UI with rounded fields and hover effects
- ⌨️ Press ESC to clear all fields instantly
- 🛡 Input validation — no crashes on invalid input
- 🔢 Binary field restricted to only 0 and 1
- 📦 Single file — 211 lines of clean code
- 🌐 Fully offline — no internet required

---

## 🚀 How to Run

### Prerequisites
- Java SE 21 or above installed

### Steps
```bash
# Clone the repository
git clone https://github.com/Kaustubhhbhoirr/UniConvert.git

# Compile
javac UnitConverter.java

# Run
java UnitConverter
```

- DocumentListener for live input
